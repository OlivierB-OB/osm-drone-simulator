# Vegetation

Vegetation in both natural and human-managed forms.

**Primary Tags:**
- `natural=*`
- `landuse=*`
- `leisure=*`
- `surface=*`

## Trees

| Tag | Definition |
|-----|-----------|
| `natural=tree` | Individual significant tree |
| `natural=tree_row` | Linear arrangement of trees between fields or along corridors |
| `natural=wood` | Forested area or woodland |
| `landuse=forest` | Managed woodland or tree plantation |
| `landuse=orchard` | Trees/shrubs cultivated for food |
| `wetland=swamp` | Forested wetland (requires `natural=wetland`) |
| `wetland=mangrove` | Salt-tolerant coastal wetland (requires `natural=wetland`) |

## Scrub - Bushes and Shrubs

| Tag | Definition |
|-----|-----------|
| `natural=shrub` | Individual bush or shrub |
| `natural=heath` | Uncultivated dwarf shrub area, shorter than scrub |
| `natural=scrub` | Uncultivated taller shrub/stunted tree area |
| `natural=shrubbery` | Actively maintained/pruned shrub area |
| `landuse=vineyard` | Grape-producing vines |
| `barrier=hedge` | Dense shrub line forming barrier |

## Herbaceous Vegetation

| Tag | Definition |
|-----|-----------|
| `natural=grassland` | Natural grass-dominated areas |
| `natural=wetland` (with subtypes) | Waterlogged areas: marsh, saltmarsh, wet_meadow, reedbed |
| `landuse=grass` | Mowed, managed grass |
| `landuse=meadow` | Pasture or meadow vegetation |

## Treeless Mixed Areas

| Tag | Definition |
|-----|-----------|
| `natural=fell` | Elevated tree-free terrain due to climate |
| `natural=tundra` | High-latitude/elevation tree-free areas |
| `landuse=greenfield` | Undeveloped areas planned for construction |
| `landuse=recreation_ground` | Open recreation areas |

## Mixed and Ornamental

| Tag | Definition |
|-----|-----------|
| `wetland=bog/fen/string_bog` | Peat-forming wetlands with mosses |
| `leisure=garden` | Decorative/scientific plant cultivation |
| `leisure=park` | Municipal green space |
| `landuse=village_green` | Village common areas |
| `landuse=greenery` | Landscaped decorative vegetation |
| `surface=grass` | Vegetated surface specification |
| `obstacle=vegetation` | Vegetation obstacles on paths |

## Tagging Notes

Avoid `landuse=field` without verification — consider alternative tags instead.
