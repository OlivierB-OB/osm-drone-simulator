# Key: `roof:*` — Roof Attributes

> Source: https://wiki.openstreetmap.org/wiki/Key:roof:shape, https://wiki.openstreetmap.org/wiki/Key:roof:colour, https://wiki.openstreetmap.org/wiki/Key:roof:material

## Description

The `roof:*` namespace documents the physical properties of a building's roof for 3D rendering. All tags require `building=*` or `building:part=*` on the same element.

---

## `roof:shape=*`

- **Element types**: area
- **Description**: The geometric type of the roof. Determines the 3D mesh generated by renderers. Defaults to `flat` when absent.
- **Status**: de facto

### Values

#### Simple pitched

| Value | Description |
|-------|-------------|
| `flat` | Level roof surface, no slope |
| `skillion` | Single sloping plane (shed roof) |
| `gabled` | Two sloping sides meeting at a central ridge |
| `half-hipped` | Gabled roof with small hipped ends at the ridge |
| `hipped` | Four sloping sides meeting at a single ridge or peak |
| `pyramidal` | Four equal triangular faces meeting at a single apex |
| `saltbox` | Asymmetric pitched roof; one slope longer than the other |

#### Complex pitched

| Value | Description |
|-------|-------------|
| `gambrel` | Two-slope roof on each side, with a steeper lower slope (barn roof) |
| `mansard` | Four sides each with a lower steep slope and upper shallow slope |
| `side_hipped` | Hipped ends on two sides only |
| `half-hipped` | Hipped lower section with gabled upper portion |
| `bellcast_gable` | Gabled with curved eaves flaring outward |
| `gabled_height_moved` | Gabled with the ridge offset from the centre |
| `crosspitched` | Multiple ridges crossing each other (T- or L-shaped plan) |

#### Curved / domed

| Value | Description |
|-------|-------------|
| `dome` | Hemispherical surface |
| `onion` | Bulbous dome, wider at mid-height than at base (Orthodox style) |
| `cone` | Conical shape, circular plan tapering to a point |
| `round` | Cylindrical curved surface (barrel roof) |
| `butterfly` | Two surfaces sloping downward toward a central valley |

#### Industrial / other

| Value | Description |
|-------|-------------|
| `sawtooth` | Repeated asymmetric triangular profile (factory rooflight) |
| `cantilever` | One end unsupported (stadium canopy) |

## `roof:height=*`

- **Element types**: area
- **Description**: The vertical rise of the roof — the distance from the top of the walls to the highest point of the roof. Measured in metres.
- **Common sub-tags**: `height=*`, `roof:shape=*`
- **Status**: de facto

---

## `roof:angle=*`

- **Element types**: area
- **Description**: The inclination of the roof slopes in degrees (0° = flat, 90° = vertical). An alternative to `roof:height=*` for specifying slope steepness.
- **Status**: de facto

---

## `roof:direction=*`

- **Element types**: area
- **Description**: The compass bearing (degrees clockwise from north, 0–360) that the roof "faces" — i.e. the direction the lower edge of the primary slope faces. Used to orient asymmetric roof shapes (skillion, saltbox, gabled).
- **Status**: de facto

---

## `roof:orientation=*`

- **Element types**: area
- **Description**: Whether the roof ridge runs along or across the longest axis of the building footprint. Supplements or replaces `roof:direction=*` for simple cases.
- **Values**: `along` (ridge parallel to longest wall), `across` (ridge perpendicular to longest wall)
- **Status**: de facto

---

## `roof:colour=*`

- **Element types**: area
- **Description**: The dominant colour of the roof surface, used by 3D renderers to texture the roof mesh. Accepts hex colour codes (`#C0392B`) or colour names (`red`, `brown`, `grey`).
- **Common sub-tags**: `roof:material=*`, `roof:shape=*`
- **Status**: de facto

---

## `roof:material=*`

- **Element types**: area
- **Description**: The outermost material of the roof surface. Renderers may use this to select a texture, optionally tinted by `roof:colour=*`.
- **Common sub-tags**: `roof:colour=*`, `roof:shape=*`
- **Status**: de facto

### Values

| Value | Description |
|-------|-------------|
| `roof_tiles` | Ceramic or terracotta tiles (the default in many regions) |
| `slate` | Natural slate tiles |
| `concrete` | Flat concrete slab or screed |
| `copper` | Copper sheet (weathers green) |
| `zinc` | Zinc sheet (grey) |
| `tin` | Corrugated tin or metal sheet |
| `eternit` | Fibre cement panels |
| `grass` | Living / turf roof |
| `glass` | Glass panels or skylights covering the entire surface |
| `tar_paper` | Bitumen felt or tar paper (flat roofs) |
| `metal` | Generic metal covering |
| `thatch` | Thatched straw or reed |
| `solar_panels` | Photovoltaic panels covering the roof |
| `wood` | Wooden shingles or planks |
| `asphalt` | Asphalt or bitumen membrane (flat roofs) |

---

## `roof:levels=*`

- **Element types**: area
- **Description**: The number of habitable levels within the roof space (e.g. attic rooms). Counted separately from `building:levels=*`.
- **Values**: Non-negative integers or decimals.
- **Status**: de facto
