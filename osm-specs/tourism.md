# Key: `tourism` — Tourism

> Source: https://wiki.openstreetmap.org/wiki/Key:tourism

## Description

The `tourism=*` key is used to tag a place or object of specific interest to tourists, covering sightseeing, accommodation, visitor information, and support facilities. It is approved for use on nodes and areas (including multipolygon relations). The key has 30 documented values and carries de facto status in the OSM community.

## Values

### `tourism=alpine_hut`

- **Element types**: node / area
- **Description**: A remote building positioned in the mountains designed to provide lodging accommodation for mountaineers, hikers, and climbers. Typically staffed and offering basic meals and sleeping facilities at high altitude.
- **Common sub-tags**: `ele=*`, `operator=*`, `capacity=*`, `opening_hours=*`, `fee=*`
- **Status**: in use
